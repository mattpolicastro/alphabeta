# alphaBeta

A serverless, browser-based decision-logging and discipline tool for empirical work. It is not a stats tool — it is a **discipline layer** that sits alongside experimentation platforms and makes implicit process tradeoffs visible. Existing tools answer "can I run this?"; alphaBeta asks "should I?" and holds you to what you committed.

**Status: pre-build.** This repository currently contains the design handoff ([`design/`](./design)) and a manifest of code earmarked from the predecessor project ([`reuse/`](./reuse)). The tech stack is being decided; application code starts when that lands.

The predecessor — a static Next.js A/B-test analysis tool — is archived at [`mattpolicastro/alphabeta-legacy`](https://github.com/mattpolicastro/alphabeta-legacy).

See [`design/README.md`](./design/README.md) for the full product overview, screen inventory, data model, and suggested build order.
